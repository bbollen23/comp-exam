Introduction
What is a Scalar Field?
Suppose we want to measure the temperature at of the surface of the Earth over a large region. We can think of this region as simply being a two-manifold ignoring the possible changes in height as we sample and only taking into consideration the geographic coordinates. Denote this space as $\mathbb{X}$. Then we also have a function $f:\mathbb{X}\to \mathbb{R}$ mapping from this two-manifold to the real numbers by associating the coordinate pair with the temperature at this point. The pair $(\mathbb{X},f)$ is called a scalar field. We can often visualize well-behaved scalar fields as some sort of mountainous or hilly terrain. Just as we can use contour lines on a map to indicate regions that are the same elevation, we are sometimes interested in a subset $U \in \mathbb{X}$ such that all points in $U$ have the same function value. These are called level sets or fibers of the scalar field Maybe something about our concern with the evolution of the level sets
Comparing Scalar Fields
Topological Evolution of a Scalar Field
Let $(\mathbb{X},f)$ be a scalar field. We can define an equivalence relation on $\mathbb{X}$ by defining $x\sim y$ if $f(x) = f(y) = a$ (they belong to the same $a$-fiber) and if $x,y$ are in the same path connected component of the $a$-fiber. Then, the quotient space $\mathbb{X}_f := \mathbb{X}/\sim_f$ is called the Reeb Graph of the scalar field $(\mathbb{X},f)$. We equip the Reeb Graph with the function $\tilde{f}$ which is defined as $\tilde{f} \circ \rho = f$, where $\rho: \mathbb{X} \to \mathbb{X}_f$ is the quotient map defined by our equivalence relation. The Reeb Graph tracks the evolution of the levelset topology of the scalar field. It shows how the fibers of the scalar field split and merge.
Categorifying Reeb Graphs and Scalar Fields
Category Theory
A category is a collection $\textbf{C}$ of objects with morphisms (also called maps) that map between these objects. Each morphism $\alpha$ has a source $A\in \textbf{C}$ and a target $B\in \textbf{C}$. Morphisms usually represent some sort of "structure" preserving idea, such as isomorphism when considering the category of vector spaces $\textbf{Vect}$, or homomorphisms in the category of groups $\textbf{Grp}$. Every object $A$ in a category $\textbf{C}$ has an identity morphsim $\textbf{id}_A:A \to A$.
A functor $\afunc{F}$ is a mapping between two categories $\textbf{C}$ and $\textbf{D}$ that satisfies the following properties:
  • For each object $A \in \textbf{C}$, there is a corresponding object $\afunc{F}(A) \in \textbf{D}$
  • For each moprhism $\alpha: A \to B$ in $\textbf{C}$, there is a corresponding morphism $\afunc{F}[\alpha]:\afunc{F}(A) \to \afunc{F}(B)$ in $\textbf{D}$.
  • The functors respect composition: $\afunc{F}[\alpha \circ \beta] = \afunc{F}[\alpha] \circ \afunc{F}[\beta]$
  • The functors respect identities: $\afunc{F}[\textbf{id}_A] = \textbf{id}_{\afunc{F}[A]}$
A functor can be thought of as a way to map between categories in the same way that functions map between spaces (vector spaces, groups, etc.)
A natural transformation a map $\eta: \afunc{F} \Rightarrow \afunc{G}$ between two functors, $\afunc{F},\afunc{G}:\textbf{C} \to \textbf{D}$. It consists of a family of maps $\eta_A:\afunc{F}(A) \to \afunc{G}(A)$ for each object $A\in \textbf{C}$, such that for each morphism $\alpha : A \to B$ in $A$, we have $\eta_B \circ \afunc{F}[\alpha] = \afunc{G}[\alpha] \circ \eta_A$. Category of Scalar Fields
We can construct the category of scalar fields (in related work also called the category of $\mathbb{R}$-spaces) denoted as $\rtop$ by stating that each object is a scalar field $(\mathbb{X},f)$ and each morphism $\alpha: (\mathbb{X},f) \to (\mathbb{Y},g)$ is a continuous, function-preserving map $\alpha:\mathbb{X}\to \mathbb{Y}$ between topological spaces. A function preserving map means that $\alpha \circ g = f$. In other words, the following diagram commutes:
We call the point pre-images $f^{-1}(a)$ of $\mathbb{X}$ the $a$-levelset or $a$-fiber of $\mathbb{X}$. In this way, we can say that the morphisms in the category of scalar fields preserve $a$-levelsets or $a$-fibers.
To make sure that we are dealing with well-behaved scalar fields, we will restrict this category to a subcategory. Let $S = \{a_1,\ldots,a_n\}$ be a finites set of real numbers, called "critical points". Then, we defined a critical fiber, $\mathbb{V}_i$, for each $0\leq i \leq n$ and a non-critical fiber, $\mathbb{E}_i$ for each $0 \leq i \leq n-1$. Each critical and non-critical fiber is defined to a be a locally path-connected, compact space.
Now we define continuous attaching maps $l_i:\mathbb{E}_i \to \mathbb{V}_i$ and $r_i:\mathbb{E}_i \to \mathbb{V}_{i-1}$ for each $0 \leq i \leq n-1$. Finally, we let $\mathbb{X}$ be the quotient space obtained by the disjoin union of $\mathbb{V}_i \times \{a_i\}$ and $\mathbb{E}_i \times [a_i,a_{i+1}]$ by making the identifications $(l_i(x),a_i) \sim (x,a_i)$ and $(r_i(x),a_{i+1}) \sim (x,a_{i+1})$ for all $i$ and all $x \in \mathbb{E}_i$. Now, define a function $f$ such that $f(x,a) = a$. We can see that the pair $(\mathbb{X},f)$ defines a scalar field since $\mathbb{X}$ is a topological space and $f$ is simply a projection onto the second factor, which is a continuous function.
Finally, we say that any scalar field that is isomorphic to another scalar field that can be constructed in the way described above is constructible. The intuition here is that between the critical points, the fibers have a disconnected "cylidrical" sctructure (or just a single cylinder). Then at the critical fibers, we have places where the cylindrical structures are able to "merge" or "split". Note again that these attaching maps are always mapping from the non-critical fibers. So if our cylinder has cylinders that split into two cylinders, there must be a critical fiber which is a figure eight, exactly at the point where they start to split. In this way, then we can always map a single cylinder to a figure eight or two cylinders from to the figure eight. We never have to worry about actually "splitting" the figure eight into two cylinders (which is discontinuous) because are maps only go twoards the critical-fibers.

Let $(\mathbb{X},f)$ be a constructible scalar field and let $(\mathbb{X}_f,\tilde{f})$ be its Reeb Graph. We begin by showing that the Reeb Graph of a constructible scalar field is another constructible scalar field and that the maps between Reeb Graphs induced by morphisms between constructible scalar fields are still morphisms.
By definition of a quotient space, we know that $\mathbb{X}_f$ is a topological space. We still must show that $\tilde{f}$ is a continuous function.
Munkres Theorem 22.2Let $\rho: X \to Y$ be a quotient map. Let $Z$ be a space and let $f: Y \to Z$ be a map that is constant on each set $\rho^{-1}(\{y\})$, for $y \in Y$. Then $f$ induces a map $g:Y to Z$ such that $g \circ \rho = f$. The induced map $g$ is continuous if and only if $f$ is continuous.
proof: We only need the backwards direction for this work. That is, we need to show that the function $g$ exists and that if $f$ is continuous then $g$ is continuous. Assume that $f$ is constant on point pre-images $\rho^{-1}(\{y\})$ and that $f$ is continuous. Then we know that $f(\rho^{-1}(\{y\}))$ is a single point in $Z$. Denote this point as $g(y)$. Then we have a map $f: X\to Z$ such that for all $x in X$, $(g \circ \rho)(x) = f(x)$. Now, let $V$ be and open set in $Z$. Then $f^{-1}(V)$ is open in $X$ and $f^{-1}(V) = \rho^{-1}(g^{-1}(V))$. Since $\rho$ is a quotient map (which means it must be continuous), $\rho^{-1}(U)$ is open when $U$ is open. Thus, $g^{-1}(V)$ must be open in $Y$ which implies that $g$ is continuous.
In our scenario, the quotient map is $\rho: \mathbb{X} \to \mathbb{X}_f$ and our continuous function is $f:\mathbb{X} \to \mathbb{R}$. Since $f$ is constant on point pre-images of $\mathbb{X}_f$, there exists a continuous function $g \circ \rho = f$. By definition of a Reeb Graph, we have that $\tilde{f} \circ \rho = f$, so $\tilde{f} = g$ and $(\mathbb{X}_f, \tilde{f})$ is a scalar field.
Let $(\mathbb{X},f)$ and$(\mathbb{Y},g)$ be constructible scalar fields and let $(\mathbb{X}_f,\tilde{f})$ and $(\mathbb{Y}_g,\tilde{g})$ be their respective Reeb Graphs. Additionally, let $\rho_f : \mathbb{X} \to \mathbb{X}_f$ and $\rho:\mathbb{Y} \to \mathbb{Y}_g$ be their respective quotient maps. Given a morphism $\alpha: \mathbb{X} \to \mathbb{Y}$ in $\rtopc$, the map $\tilde{\alpha}$ defined by $\tilde{\alpha} \circ \rho_f = \rho_g \circ \alpha$ is a morphism in $\rtopc$.
To show that this map $\tilde{\alpha}$ is a morphism in $\rtopc$, we must show that it is a continuous, function-preserving map. Consider the composition of maps $(\rho_g \circ \alpha): \mathbb{X} \to \mathbb{Y}_g$. This map is continuous since both $\rho_g$ and $\alpha$ are continuous. Now, consider a point pre-image $\rho_f^{-1}(\{y\}) \in \mathbb{X}$ where $\tilde{f}(y) = a$. Then this pre-image is the $a$-fiber of $\mathbb{X}$. Since $\alpha$ is a morphism in $\rtopc$, it preservers level sets, so $\alpha(\rho_f^{-1}(\{y\}))$ is the $a$-fiber of $\mathbb{Y}$. Then, since $\rho_g$ is the quotient map turning all $a$-fibers into a single contracted point in $\mathbb{Y}_g$, the composition of maps $(\rho_g \circ \alpha)$ is constant on pre-images of $rho_f$. Using Munkres Theorem 22.2 again, we have that the induced map $\tilde{\alpha}$ in the equation $\rho_g \circ \alpha = \tilde{\alpha} \circ \rho_f$ must be continuous, since $\rho_g \circ \alpha$ is continuous and constant on the point pre-images of the quotient map $\rho_f$.
To show that $\tilde{\alpha}$ is a function preserving map, we need to show that $\tilde{f} = \tilde{g} \circ \tilde{\alpha}$. We begin with the fact that $f = g \circ \alpha$: \begin{equation} f = g \circ \alpha \hspace{0.1in} \Rightarrow (\tilde{f} \circ \rho_f) = (\tilde{g} \circ \rho_g) \circ \alpha \hspace{0.1in} \Rightarrow (\tilde{f} \circ \rho_f) = \tilde{g} \circ (\rho_g \circ \alpha) \hspace{0.1in} \Rightarrow (\tilde{f} \circ \rho_f) = \tilde{g} \circ (\tilde{\alpha} \circ \rho_f) \hspace{0.1in} \Rightarrow \tilde{f} = \tilde{g} \circ \tilde{\alpha} \end{equation}
